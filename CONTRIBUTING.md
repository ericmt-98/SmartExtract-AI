# Guía de Contribución

¡Gracias por interesarte en mejorar este proyecto! Como este sistema utiliza una arquitectura híbrida con IA local, cualquier mejora en la precisión o eficiencia es bienvenida.

## ¿Cómo contribuir?

1.  **Reportar Errores:** Si encuentras un fallo, abre un "Issue" describiendo el problema y adjuntando (si es posible) un ejemplo del texto que causó el error.
2.  **Sugerir Mejoras:** Si tienes ideas para mejorar las Regex o el manejo del LLM, ¡somos todo oídos!
3.  **Pull Requests:** 
    - Crea un fork del repositorio.
    - Crea una rama para tu mejora (`git checkout -b feature/MejoraIncreible`).
    - Asegúrate de que el código siga las guías de estilo PEP 8.
    - Envía tu Pull Request.

## Áreas de Interés
- Optimización de promts para Ollama.
- Nuevos patrones Regex para diferentes tipos de facturas internacionales.
- Mejora en el pre-procesamiento de imágenes con OpenCV.

---
¡Juntos podemos hacer la extracción de datos más inteligente y privada!
